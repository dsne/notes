# Atom 패키지

## project-manager
- 설정
  - Project View에 저장할 프로젝트 상태 실행
  - Packages - Project Manager - Save Project
  - 이름 지정
- 실행
  - `Ctrl + Command + P`
- 수정
  - 저장 정보 수정
    - Packages - Project Manager - Edit Project
  - 프로젝트 정보 수정
    - Packages - Project Manager - Edit Projects

## tree-view
- 무시할 파일 숨기기
  - Preferences - Settings - Packages
  - tree-view 검색
  - Hide Ignored Names 선택
